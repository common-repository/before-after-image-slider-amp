== Description ==

This plugin makes it easy to create a before and after image comparison slider, using <a href="https://amp.dev" target="_blank">AMP</a> technology. It's a quick method of implementing the related <a href="https://amp.dev/documentation/examples/components/amp-image-slider/" target="_blank">amp-image-slider</a> component on your site. 

For best results use the <a href="https://wordpress.org/plugins/amp/" target="_blank">official AMP plugin</a>, with the viewing options applicable with this official plugin, rendering on AMP or non AMP URLs.

<strong>Features: </strong>

<ul>
	<li>Easy to setup, select your images, width and height</li>
	<li>Super Lightweight</li>
	<li>Responsive (via the AMP intrinsic layout)</li>
	<li>Shortcode activated</li>
	<li>Fully AMP compatible</li>
	<li>Powered by AMP technology, optimized for performance</li>
</ul>

View a demo <a href="https://jamesozz.ie/1356/before-and-after-image-slider/">here.</a>

== Installation ==
<strong>Using The WordPress Dashboard</strong>

1. Install as a typical WordPress plugin (from the WordPress repository or via a file upload)
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Once activated you'll find the configuration options within the settings tab under "Settings > Before After Image Slider (AMP) "

== Frequently Asked Questions ==

= Does this plugin come with settings? =

There are width and height settings configurable from the plugin settings. Get in touch if you'd like more features, or check out the related <a href="https://amp.dev/documentation/examples/components/amp-image-slider/" target="_blank">AMP component</a> if you'd like to configure yourself. 

= Who's this plugin suitable for? =

For anybody looking to have images which can be easily measured against other images. In particualar the following may be useful:
<ol>
	<li>Beaticians</li>
	<li>Builders or tradesmen</li>
	<li>Dentists</li>
	<li>Car Wash Operators</li>
	<li>Painters</li>
	<li>Fitness Coaches / Gyms</li>

</ol>

= Can I create multiple before and after images? =

At present no, you can only use the same 2 images. As it's shortcode activated you can use the before and after slider on multiple pages or posts, but it will be the same 2 images.   

= It's not working, whats wrong? =

Did you check the configuration options to ensure you have it working on the correct version of your site, AMP or non AMP? If so check that your shortcode was implemented correctly. The correct shortcode to use is [jozz-ampimageslider]

You may also want to check and ensure you've selected images both in the before and after fields.
